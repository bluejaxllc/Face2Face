# Face 2 Face — KPI Framework & Metrics Benchmarks

## 1. CORE ENGAGEMENT METRICS

| Metric | Average | Good | Great | Exceptional |
|---|---|---|---|---|
| DAU/MAU Ratio | 15-20% | 20-25% | 25-35% | 40%+ (Grindr ~50%) |
| Session Duration | 13.21 min | 10-15 min | 15-20 min | 20+ min |
| Sessions/Day | 2-3x | 4x (Tinder avg) | 5-6x | 7+ |
| Messages/User/Day | Skewed | 3-5 | 5-10 | 10+ |
| Profile Completion | ~15% | 40% | 55% | 70%+ |
| Photo Upload | 2-3 avg | 4 photos | 5-6 photos | 6+ |

### Funnel Benchmarks
- Install → Profile Creation: 60-75%
- Profile → Active Browsing: 50-70%
- Browse → First Match (7 days): 30-60%

## 4. MONETIZATION METRICS

| Metric | Benchmark |
|---|---|
| ARPU | $0.70-$5.00/month |
| ARPPU | $15-$25/month |
| Free → Paid Conversion | 3-8% (new: 1-3%, optimized: 5-8%) |
| LTV:CAC Ratio | 3:1 minimum healthy |
| CAC (Tier 1 Markets) | $4-$25/user |
| CAC (LATAM) | $0.50-$3.00 |
| Payback Period | Target: 5-7 months |
| Monthly Churn | >20% MoM |

### LTV Formula
LTV = (ARPU × Gross Margin %) / Monthly Churn Rate
- At 20% churn → 5-month avg lifespan
- At 10% churn → 10-month avg lifespan

## 5. LOCATION-SPECIFIC KPIs (F2F Unique)

### Map Density Metrics
- Active Users per Square Mile — most critical
- First-Session Match Rate — if low, density too thin
- Regional Cohort Retention — high-density retains better

### Venue/Hotspot Metrics
- Venue Check-in Rate
- Active Users per Venue
- Proximity "Bump" Rate
- First-Interaction-at-Venue Rate

### Time-of-Day Activity
- Social: Evening (7-11 PM)
- Business: Daytime (9 AM-5 PM)
- Love: Late evening (8 PM-midnight)
- Weekend peaks for Social/Love; Weekday for Business

### F2F-Specific KPIs
1. Map Interaction Rate (taps on pins per session)
2. Pillar Switching Rate
3. Venue Discovery Rate
4. Proximity Alert Engagement
5. Map-to-Chat Conversion
6. Real-World Meetup Rate

## 6. SAFETY & TRUST METRICS

| Metric | Benchmark |
|---|---|
| Harassment Rate | ~25% report some form (1 in 4) |
| Identity Verification Rate | Avg ~28%, high-verification: 90%+ |
| Bot Block Rate | Best: 99% at registration |
| Report Response Time | Best: <1 minute |
| Moderation Latency | Top tier: <200ms |
| Detection Accuracy | Advanced AI: up to 99.9% |

Key: 90%+ verification → harassment at 1/3 the rate

## 7. NORTH STAR METRIC

### Recommended: "Real-World Meetups Facilitated per Day"
- Aligns with F2F's DNA (digital-to-physical)
- Spans all 3 pillars
- Drives retention, word-of-mouth, monetization

### Supporting Metrics Tree
```
North Star: Real-World Meetups/Day
├── Leading: Map Interactions per Session
├── Leading: Messages Sent per User
├── Leading: Venue Check-in Rate
├── Leading: Proximity "Bump" Rate
├── Lagging: D7/D30 Retention
├── Lagging: NPS / User Satisfaction
└── Lagging: LTV / Revenue
```
