Mobile Capacitor Android

🔴 6 Critical Architecture Gaps

GAP 1 No PostGIS

Nearby query = full table scan + JS filtering O(n). PostGIS = spatial index O(log n).

GAP 2 Base64 Photos in DB

No CDN. Moving photos out = 80%+ DB size reduction.

GAP 3 No WebSocket Broadcast

ws dependency exists but NOT wired up. Clients must poll.

GAP 4 No Redis Cache

No caching layer for hot geospatial queries.

GAP 5 No Spatial Indexing

No GiST index, no geohashing, no H3 hexagons.

GAP 6 Session-Cookie Auth

Won't work for native mobile apps. Migrate to JWT.
